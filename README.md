# Spring Boot Web

Projeto que tinha como o objetivo criar um CRUD para manipulação de cartórios(ID e Nome)


## Funções presentes
- Criar cartório
- Editar cartório
- Excluir cartório
- Listar cartório
- Vizualizar cartório

## Tecnologias utilizadas
### Back End
- Spring Boot
- JPA/Hibernate
- PostgreSQL

### Front End
- JSP
- Jquery
- Bootstrap
